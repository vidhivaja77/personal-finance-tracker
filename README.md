# 💰 Personal Finance Tracker Dashboard

A full-stack **MERN Stack** web application to help users track their 
income and expenses with an interactive dashboard, AI-powered insights, 
and real-time data visualization.

## ✨ Features

- 📊 **Real-time Dashboard** with interactive Pie Chart & Bar Chart
- 🤖 **AI Insights** — smart suggestions based on spending patterns
- 📋 **Transaction Management** — add, edit and delete entries
- 📄 **PDF Export** — download transaction history as PDF
- 💹 **Income vs Expense** overview with visual analytics
- 🔄 Real-time data updates via RESTful APIs
- 📱 Fully Responsive UI for all devices
- 🗄️ MongoDB for secure and dynamic data storage

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js, HTML5, CSS3 |
| Backend | Node.js, Express.js |
| Database | MongoDB |
| Charts | Pie Chart, Bar Chart (Real-time) |
| Other | REST APIs, CRUD Operations, PDF Export, Git |

## 📁 Project Structure

**Frontend**
```
src/
├── components/    # Reusable React components
├── pages/         # Dashboard, Transactions pages
└── App.js         # Main app entry point
```

**Backend**
```
├── config/        # Database configuration
├── controllers/   # Business logic
├── models/        # MongoDB schemas
├── routes/        # API endpoints
└── server.js      # Backend entry point
```

## 🚀 How to Run Locally
```bash
# Clone the repository
git clone https://github.com/vidhivaja77/personal-finance-tracker

# Run Frontend
cd frontend
npm install
npm start

# Run Backend
cd backend
npm install
npm start
```
